# ФотоКопиЦентр — новый сайт (v2, весь сайт)

Рабочий прототип: реальный сервер на Node.js принимает и сохраняет заказы,
файлы, отзывы и заявки на звонок. Без npm-зависимостей — запускается сразу.

## Запуск

```
node server.js
```

Откройте http://localhost:3000

Сайт также можно открыть напрямую двойным кликом по `public/index.html` —
дизайн и все интерактивные штуки (фоторедактор, конструктор печатей, каталог)
будут работать, а вот отправка заказов/отзывов/звонков будет в демо-режиме,
пока не запущен сервер.

Заказы сохраняются в `data/orders.json`, файлы — в `uploads/`, заявки на
звонок — в `data/callbacks.json`, отзывы с сайта (на модерации) — в
`data/reviews-inbox.json`. Акции для главной страницы редактируются в
`data/promos.json` (эти же тексты потом можно использовать в рекламе
Яндекс.Директа).

## Структура сайта

- `index.html` — главная (акции, преимущества, услуги, отзывы, карта)
- `o-kompanii.html` — о компании
- `uslugi.html` — полный каталог услуг с фильтром по категориям + офлайн-услуги
- `galereya.html` — галерея работ с фильтром
- `otzyvy.html` — все отзывы + форма добавления отзыва
- `kontakty.html` — контакты, карта, форма обратного звонка

Шапка, подвал и все модалки (заказ, звонок, офлайн-услуга, фоторедактор,
конструктор печатей) вынесены в `public/js/layout.js` — один источник правды
для всех страниц.

## Что нужно от вас, чтобы двигаться дальше

1. **Логотип.** Сейчас подключен напрямую с текущего сайта
   (`https://sap-uslugi.ru/img/logo.png`). Для продакшена лучше прислать
   файл в хорошем качестве (PNG/SVG).
2. **Ссылка на мессенджер MAX** — сейчас стоит заглушка `#`.
3. **Онлайн-оплата.** Реальный приём платежей требует подключения провайдера
   (ЮKassa, CloudPayments, Т-Касса и т.п.) — нужны реквизиты продавца и
   API-ключи. Сейчас на сайте есть кнопка «Оплатить онлайн», которая создаёт
   заявку, но платёж демонстрационный (это явно видно на кнопке — «Демо
   онлайн-оплаты»), пока вы не пришлёте данные провайдера.
4. **Фото для галереи.** Сейчас в галерее временно стоят фото с текущего
   сайта sap-uslugi.ru — пришлите свежие примеры работ, заменим за 5 минут
   (файл `public/js/gallery-data.js`).
5. **Макеты фотокерамики/табличек** — вы упоминали, что можете прислать фото
   образцов макетов, размеров и материалов — добавим отдельную галерею макетов.
6. **Хостинг и домен** для продакшн-версии (сейчас сайт работает локально).
